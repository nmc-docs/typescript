# Giới thiệu

- **Type Operator** trong Typescript là các phép toán được sử dụng để tạo ra các kiểu dữ liệu mới từ các kiểu dữ liệu hiện có
- Các type operator phổ biến hay dùng:
  - [keyof](./keyof)
  - [typeof](./typeof)
  - [Indexed Access Types](./indexed-access-types)
